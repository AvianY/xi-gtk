## Shortcuts

Shortcut                                         | Command
-------------------------------------------------|---------
<kbd>Control</kbd>+<kbd>N</kbd>                  | New File
<kbd>Control</kbd>+<kbd>O</kbd>                  | Open File
<kbd>Control</kbd>+<kbd>S</kbd>                  | Save
<kbd>Control</kbd>+<kbd>Shift</kbd>+<kbd>S</kbd> | Save As
